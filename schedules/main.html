Weekly problem sets are listed separately, as are WeBWorK problems.

To import a Sage worksheet into SageMathCloud, first download the worksheet file to your local machine from a link below. Then, open a new Sage Worksheet. Drag the downloaded file right onto the Worksheet pane and it should come right up. Then you can modify your local copy in the cloud and it will be synced and saved automatically.
